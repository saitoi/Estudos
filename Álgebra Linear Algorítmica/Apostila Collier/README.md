# Instruções para transferir as anotações para o <ins>seu</ins> Obsidian
***

1. Ao instalar o Obsidian, o seu computador terá uma pasta `Obsidian` criada em algum local da área de trabalho.
2. Baixe meu repositório completo `saitoi/Estudos` para o seu computador.
3. Insira o repositório baixado na pasta `Obsidian` e pronto. Aproveite minhas anotações..
