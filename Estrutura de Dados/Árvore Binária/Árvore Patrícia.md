# Árvore Patrícia

> $\textit{Definição.}$ Estrutura de dados do tipo árvore ordenada derivada da [[Árvore Digital (Trie)|árvore digital]] de modo que todos os nós possuem nenhum ou dois descendentes pois **agrupamos** um grupo disperso de nós.
