# Casting

> $\textit{Definição.}$ Processo de converter um valor de um tipo de dado em outro tipo de dado compatível. Necessário quando se deseja realizar operações com dados de diferentes tipos.

Em Java, os tipos padrões para valores inteiros e decimais é <ins>int</ins> e <ins>double</ins>, respectivamente. Assim sendo, é necessário realizar casting para certificar que as expressões estão recebendo os valores formatados corretamente.
