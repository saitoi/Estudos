# Unicode UTF-16

> $\textit{Definição.}$ Diferentemente de C, a linguagem Java utiliza a codificação de caracteres Unicode UTF-16 que abrange uma gama maior de símbolos (+ 1.000.000) em relação ao [[Tabela ASCII|ASCII]].

==Colocar parte abaixo em [[Variáveis e Primitivos|Variáveis e Primitivos]].==

Portanto, ao utilizar o tipo primitivo `char` podemos realizar a declaração de três formas distintas:

|Tipo de Atribuição|Exemplo Código|
|-|-|
|Caractere literal|`char myChar = 'D'`|
|Valor Unicode|`char myChar = '\u0044'`|
|Valor inteiro|`char myChar = 68`|
