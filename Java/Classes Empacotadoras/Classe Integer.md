# Classe Integer

> $\textit{Definição.}$ [[Classes Empacotadoras|Classe empacotadora]] que permite tratar dados do tipo `Integer` como objetos.

### $\textup{Principais métodos estáticos e de instância}$

|Método|Descrição|Tipo|
|---|---|---|
|`parseInt(String s)`|Converte uma string em um inteiro.|Estático|
|`parseInt(String s, int radix)`|Converte uma string em um inteiro usando uma base específica.|Estático|
|`valueOf(int i)`|Retorna um objeto `Integer` que representa o valor inteiro `i`.|Estático|
|`byteValue()`|Retorna o valor do objeto como um byte.| Instância|
|`shortValue()`|Retorna o valor do objeto como um short.| Instância|
|`intValue()`|Retorna o valor do objeto como um int.| Instância|
|`longValue()`|Retorna o valor do objeto como um long.| Instância|
|`floatValue()`|Retorna o valor do objeto como um float.| Instância|
|`doubleValue()`|Retorna o valor do objeto como um double.| Instância|
|`equals(Object obj)`|Verifica se o objeto é igual ao outro objeto.| Instância|
|`compareTo(Integer anotherInteger)`|Compara este objeto com outro objeto Integer.| Instância|
|`toString()`|Retorna uma representação em string do objeto Integer.| Instância|
|`hashCode()`|Retorna um código de hash para o objeto.| Instância|
|`getClass()`|Retorna a classe do objeto.| Instância|
