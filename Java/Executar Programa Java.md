# Executar Programa Java

Para executar um programa em Java por meio da linha de comando, trataremos dos seguintes passos:

1. **Compilação**
	- Para compilar o programa desejado, devemos entrar na aplicação do Terminal e navegar até o diretório do arquivo por meio do comando "change directory" `cd`, aqui está um exemplo: `cd Documents/codigo/Java`.
	- Posto isso, a compilação é feita da seguinte forma: `javac programa.java -flags`.

2. **Execução**
	- Por fim, para executar o programa empregamos a seguinte notação: `java programa`, arquivo principal sem a extensão `.java`.
