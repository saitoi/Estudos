# Jshell

> $\textit{Definição.}$ Ferramenta interativa de linha de comando em Java que permite escrever e executar rapidamente trechos de código Java com feedback imediato sobre os resultados **sem a necessidade de escrever um programa completo**.

Inicializamos o Jshell digitando `jshell` na linha de comando do terminal. Posto isso, podemos procurar suas funcionalidades segunda a lista de comandos abaixo:

| Comando                 | Descrição                                                           |
|:----------------------- | ------------------------------------------------------------------- |
| `/help` ou `/h`         | Exibe a lista de comandos de ajuda disponíveis no JShell.           |
| `/exit` ou `/quit`      | Sai do ambiente JShell.                                             |
| `/vars` ou `/variables` | Lista todas as variáveis definidas no ambiente JShell.              |
| `/methods`              | Lista todos os métodos definidos no ambiente JShell.                |
| `/types`                | Lista todos os tipos definidos no ambiente JShell.                  |
| `/imports` ou `/i`      | Lista todos os imports atuais.                                      |
| `/history`              | Exibe o histórico de comandos que você digitou.                     |
| `/reset`                | Reinicia o ambiente JShell, removendo definições.                   |
| `/save [filename]`      | Salva comandos digitados até o momento em um arquivo.               |
| `/open [filename]`      | Abre e executa um arquivo de script JShell.                         |
| `/edit [id]`            | Abre o editor para editar um comando pelo seu ID.                   |
| `/drop [id]`            | Remove um comando específico pelo seu ID.                           |
| `/vars [varName]`       | Exibe informações detalhadas sobre uma variável específica.         |
| `/methods [methodName]` | Exibe informações detalhadas sobre um método específico.            |
| `/types [typeName]`     | Exibe informações detalhadas sobre um tipo específico.              |
| `/list`                 | Lista todos os comandos definidos no ambiente Jshell ("Histórico"). |
