# Limites

> $\textit{Definição.}$ Limites de armazenamento de cada tipo de variável, expresso por constantes definidas na [[Código C/Conceitos Gerais/Bibliotecas#^f69ec1| biblioteca limits.h ]].

|  Constante   | Descrição                            |
|:------------:| ------------------------------------ |
|  `CHAR_BIT`  | Número de bits em um `char`          |
| `SCHAR_MIN`  | Valor mínimo de `signed char`        |
| `SCHAR_MAX`  | Valor máximo de `signed char`        |
| `UCHAR_MAX`  | Valor máximo de `unsigned char`      |
|  `CHAR_MIN`  | Valor mínimo de `char`               |
|  `CHAR_MAX`  | Valor máximo de `char`               |
|  `SHRT_MIN`  | Valor mínimo de `short`              |
|  `SHRT_MAX`  | Valor máximo de `short`              |
| `USHRT_MAX`  | Valor máximo de `unsigned short`     |
|  `INT_MIN`   | Valor mínimo de `int`                |
|  `INT_MAX`   | Valor máximo de `int`                |
|  `UINT_MAX`  | Valor máximo de `unsigned int`       |
|  `LONG_MIN`  | Valor mínimo de `long`               |
|  `LONG_MAX`  | Valor máximo de `long`               |
| `ULONG_MAX`  | Valor máximo de `unsigned long`      |
| `LLONG_MIN`  | Valor mínimo de `long long`          |
| `LLONG_MAX`  | Valor máximo de `long long`          |
| `ULLONG_MAX` | Valor máximo de `unsigned long long` |
