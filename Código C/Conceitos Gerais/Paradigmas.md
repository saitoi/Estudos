# $\text{Paradigmas}$

Trata de diferentes abordagens ou estilos de programação para resolução de problemas de forma estruturada. Os dois principais são:

- $\texttt{Comparação e troca}$: Paradigma na ordenação em que os elementos são comparados dois a dois e são trocados, caso necessário, para se obter a ordem desejada. ^3d3eca

- $\texttt{Divisão e conquista}$: Paradigma na ordenação em que o problema principal é dividido em subproblemas menores e mais simples, resolvendo-os de forma recursiva e, em seguida, combinando as soluções. ^ddb8ae
