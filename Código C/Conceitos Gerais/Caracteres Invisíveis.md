# Caracteres Invisíveis

> $\textit{Definição.}$ Caracteres com funcionalidades específicas os quais iniciam com contra barra `\`.

| Caracteres  | Significado | Utilidade |
|:-:|-|-|
| `\n` | Newline | Quebra linha  |
| `\t` | Tab | Move o cursor para próxima tabulação |
| `\b` | Backspace | Retorna um caractere |
| `\f` | Feed Forward | Salta uma página |
| `\r` | Carriage Return | Posiciona cursor no início da linha |
| `\a` | Alerta | Campainha do sistema |
| `\0` | Null | Caractere que termina [[Código C/Conceitos Gerais/Strings]] |
