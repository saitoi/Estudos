# Código de Formatação

| Código de Formatação | Descrição                                             |
|:--------------------:| ----------------------------------------------------- |
|         `%d`         | Inteiro decimal (formato padrão)                      |
|         `%f`         | Número de ponto flutuante (formato padrão)            |
|         `%c`         | Caractere                                             |
|         `%s`         | String                                                |
|        `%lf`         | Double (ponto flutuante de precisão dupla)            |
|         `%p`         | Endereço de memória (ponteiro)                        |
|         `%u`         | Inteiro sem sinal (formato decimal)                   |
|     `%x` ou `%X`     | Inteiro hexadecimal (letras minúsculas ou maiúsculas) |
|         `%o`         | Inteiro octal                                         |
|        `%lld`        | Long long decimal (inteiro longo longo)               |
|        `%llu`        | Long long decimal sem sinal                           |
|        `%Lf`         | Long double (ponto flutuante de precisão estendida)   |
|     `%e` ou `%E`     | Notação científica                                    |
|     `%g` ou `%G`     | `%e` ou `%f`, dependendo da magnitude                 |
|        `%.*f`        | Especifica a precisão decimal para ponto flutuante    |
