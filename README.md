
# Instruções para transferir minhas anotações para o <ins>seu</ins> Obsidian

1. Ao instalar o Obsidian, o seu computador criará uma pasta `Obsidian` em algum local da área de trabalho.
2. Baixe meu repositório completo `saitoi/Estudos` para o seu computador.
3. Insira o repositório baixado (todos os arquivos) na pasta `Obsidian` e pronto. Aproveite minhas anotações.

> As configurações do <ins>meu</ins> Obsidian podem ser encontradas na pasta `.obsidian`.

# Como usar o Obsidian?

Primeiramente, devemos ter em mente que o Obsidian trata, em sua maioria, de arquivos no formato **Markdown** com extensão `.md`, ou seja, você não conseguirá acessar arquivos `.txt` ou alguns outros formatos.

Aqui está uma lista dos tipos de arquivos compatíveis no Obsidian:

| Imagem | Vídeo  |  Docs  |
|:------:|:------:|:------:|
| `.png` | `.mp4` | `.pdf` |
| `.jpg` |        |        |
| `.svg` |        |        |
