
# Instruções para transferir as anotações para o <ins>seu</ins> Obsidian

1. Ao instalar o Obsidian, o seu computador criará uma pasta `Obsidian` em algum local da área de trabalho.
2. Baixe meu repositório completo `saitoi/Estudos` para o seu computador.
3. Insira o repositório baixado (todos os arquivos) na pasta `Obsidian` e pronto. Aproveite minhas anotações.

> As configuraçõe do <ins>meu</ins> Obsidian podem ser encontradas na pasta `.obsidian`.

# Como usar o Obsidian?

Primeiramente, o Obsidian trabalha com arquivos no formato **Markdown** com extensão `.md`, ou seja, você não conseguirá acessar arquivos `.txt` ou alguns outros formatos.
